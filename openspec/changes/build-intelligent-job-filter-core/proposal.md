# 提案 (Proposal): 智能化招聘信息检索与筛选工具 (TS 版)

## 1. 核心价值 (Core Value)

构建一个基于 DeepSeek API 联网检索与语义化提取的智能招聘信息筛选器，实现高效率、低维护成本的岗位检索。

## 2. 核心功能点 (Key Features)

- **联网检索与语义提取**: 采用搜索引擎 + LLM 提取模式 (ADR 0003)，无需传统爬虫维护，精准提取核心字段。
- **输入参数约束 (Input Constraints)**:
  - **求职城市 (Required)**: 严格执行地理位置过滤。
  - **岗位/技能 (Required)**: 岗位类型或核心技能，两者必选其一。
  - **薪资要求 (Optional)**: 设定最低薪资门槛。
- **智能化去重 (ADR 0002)**: 基于 `(公司名 + 职位名 + 地点)` 唯一指纹，结合 30 天 TTL 与内容比对逻辑，实现高效增量检索。
- **结构化输出**: 生成 `[YYYYMMDD]_[搜索条件].md` 格式的 Markdown 表格，单次 20 个岗位，追加写入模式。
- **交互流控制**: 完成单轮（20 个）搜索后系统必须暂停并交互询问是否继续。

## 3. 验收标准 (Acceptance Criteria)

- [ ] 系统基于 `seen_jobs.json` 索引库实现精准去重与过期岗位重检索。
- [ ] 采用 LLM 联网检索模式，输出字段包含：岗位类型、薪资、行业、企业类型、人员规模、技能要求。
- [ ] 输出文件名符合命名规范且格式为 Markdown 表格。
- [ ] 实现分页确认的阻塞交互逻辑 (Node.js Readline)。
